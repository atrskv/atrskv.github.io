### Тестирование 

Теория *(принципы, уровни и т.д.)*, тест-анализ и тест-дизайн *(эквивалентное разбиение, граничные значения, таблица решений, состояние и переходы, комбинаторные техники)*, исследовательское тестирование *(туры Уиттакера, чит-листы, SBTM и TBTM)*

### Автоматизация

**Языки:** Python, Swift, SQL

**UI *(web)*:** Selene, Playwright, свой враппер поверх Selenium WebDriver
*(Page Object Model и её вариации: Element/Steps/Fluent Page Object, строю стабильные CSS-  и Xpath-селекторы, умею писать DSL-обертки над XPath)* с целью повышения читаемости и переиспользования кода

**API**: REST API *(OpenAPI/Swagger)*, gRPC *(Protobuf)*, SOAP *(WSDL/XSD)*, Requests, Pydantic, Postman

**Mobile**: Нативные автотесты с XCTest для iOS, кросс-платформенные с Appium *(Screen Object Model, запуск на симуляторах и реальных устройствах)*
